Marvel Comics Database

A database for cataloging characters, events, locations, superpowers, and teams of the Marvel Comics universe.

Follow the links in the nav bar to go to the following pages:

  • Search/Add: On this page you can select a table to view, search, and add indiviual entities to.
  • Update: On this page you can select a table to view and update individual entities.
  • Delete: On this page you can select a table to view and delete individual entities.
  • M-to-M: On this page you can select a many-to-many relationship to view, add and remove individual relationships.
